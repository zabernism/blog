LinkedList

张帅
javajava基础集合

发布于：Jul 27, 2021
次浏览

现在栈或者队列的首选是ArrayDeque。

总体介绍

LinkedList同时实现了List接口和Deque接口，允许null元素。

LinkedList底层通过双向链表实现。节点用内部类Node表示,通过 first 和 last 引用分别指向链表的第一个和最后一个元素。链表为空时 first 和 last 指向 null 。

LinkedList 没有实现同步，如果需要多个线程并发访问，可采用 Collections.synchronizedList() 包装。

方法剖析

add()

add(E e)

在 LinkedList 的末尾插入元素

add(int index, E element)

  1. 先根据 index 找到要插入的位置；
  2. 修改引用，完成插入操作。

remove()

remove(Object o)

删除跟指定元素相等的第一个元素

  1. 根据 equals 找到要删除元素的引用
  2. unlink(Node x)，完成删除操作。

remove(int index)

删除指定下标处的元素

  1. 根据 index 找到要删除元素的引用
  2. unlink(Node x)，完成删除操作。

unlink(Node x)

get、set

get(int index)

通过node(int index)得到

set(int index, E element)

通过node(int index)找到，修改

node(int index)

因为链表双向，node(int index) 中取决于条件 index < (size >> 1)决定朝那个方向找。

总结

  • LinkedList 插入，删除都是移动指针效率很高。
  • 查找需要进行遍历查询，效率较低。
