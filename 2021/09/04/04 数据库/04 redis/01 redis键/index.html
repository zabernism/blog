Redis keys 命令

[TOC]

DEL key 该命令用于在key存在时返回删除key

用于删除已存在的键。不存在的 key 会被忽略。

语法

redis DEL 命令基本语法如下：

DEL KEY_NAME

返回值

被删除 key 的数量。

基本使用

DUMP key 序列化给定 key ，并返回被序列化的值。

用于序列化给定 key ，并返回被序列化的值。

语法

DUMP KEY_NAME

返回值

如果 key 不存在，那么返回 null 。 否则，返回序列化之后的值。

基本使用

key存在

key不存在

EXISTS key 检查给定 key 是否存在。

用于检查给定 key 是否存在。

语法

EXISTS KEY_NAME

返回值

若 key 存在返回 1 ，否则返回 0 。

基本使用

key存在

key不存在

EXPIRE key seconds 为给定 key 设置过期时间，以秒计。

用于设置 key 的过期时间，key 过期后将不再可用。单位以秒计。

语法

Expire KEY_NAME TIME_IN_SECONDS

返回值

设置成功返回 1 。 当 key 不存在或者不能为 key 设置过期时间时返回 0 。

基本使用

设置key 10秒之后过期

PEXPIRE key milliseconds 设置 key 的过期时间以毫秒计。

用于设置 key 的过期时间，以毫秒计。key 过期后将不再可用。

语法

PEXPIREAT KEY_NAME TIME_IN_MILLISECONDS_IN_UNIX_TIMESTAMP

返回值

设置成功返回 1 。 当 key 不存在或者不能为 key 设置过期时间时返回 0 。

基本使用

KEYS pattern 查找所有符合给定模式( pattern)的 key 。

用于查找所有符合给定模式 pattern 的 key 。。

语法

KEYS PATTERN

返回值

符合给定模式的 key 列表 (Array)。

基本使用

获取redis中所有key

MOVE key db 将当前数据库的 key 移动到给定的数据库 db 当中。

用于将当前数据库的 key 移动到给定的数据库 db 当中。

语法

redis Move 命令基本语法如下：

MOVE KEY_NAME DESTINATION_DATABASE

返回值

移动成功返回 1 ，失败则返回 0 。

基本使用

key存在时候

当两个数据库中都存在这个key时候

PERSIST key 移除 key 的过期时间，key 将持久保持。

用于移除给定 key 的过期时间，使得 key 永不过期。

语法

PERSIST KEY_NAME

返回值

当过期时间移除成功时，返回 1 。 如果 key 不存在或 key 没有设置过期时间，返回 0 。

PTTL key 以毫秒为单位返回 key 的剩余的过期时间。

TTL key 以秒为单位，返回给定 key 的剩余生存时间(TTL, time to live)。

RANDOMKEY 从当前数据库中随机返回一个 key 。

RENAME key newkey 修改 key 的名称

RENAMENX key newkey仅当 newkey 不存在时，将 key 改名为 newkey 。

SCAN cursor [MATCH pattern] [COUNT count] 迭代数据库中的数据库键。

TYPE key返回 key 所储存的值的类型。
